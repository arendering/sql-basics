===============================
=== 6. Поиск с помощью LIKE ===
===============================

LIKE без доп. параметров работает как оператор '='
Для строк не учитывается регистр
Используем для небольших по объему полях для простого поиска.


*) LIKE BINARY - поиск с учётом регистра
  SELECT * FROM users WHERE first_name LIKE BINARY 'Наталья';

*) спец. символы - % и _
  % - любое количество(в т.ч и ноль) и качество символов
  SELECT * FROM users WHERE passport LIKE '3200%11'

  _ - ровно один символ


LIKE может работать с типами, отличными от строк
пример:
  SELECT * FROM users WHERE id LIKE '_';  - поиск записей, у которых id состоит из одной цифры



--- Полнотекстовый поиск ---

Для поиска в больших по объёму полях и сложном поиске используем полнотекстовый поиск.

Для поиска нужно создать полнотекстовый индекс по полю, по которому будем искать.
 CREATE FULLTEXT INDEX idx_name ON products(name);

Полнотекстовый индекс - это словарь, в котором ключ - это слово, а значения - массив цифр, указывающих номер колонки, где содержится слово.


SELECT * FROM products
WHERE MATCH(name) AGAINST ('микроволновая печь') - ищем слова по отдельности. Сортировка идёт по релевантности содержимого - т.е чем
                                                   точнее содержимое подпадает под фразу, тем выше будет находится запись в результирующей таблице.


SELECT * FROM products
WHERE MATCH(name) AGAINST ('+микроволновая печь' IN BOOLEAN MODE) - boolean mode - специальный логический режим, который позволяет создавать более точные
                                                                   и гибкие запросы.
                                                                   + - слово обязательно должно присутствовать в поле поиска
                                                                   - - слово должно отсутствовать в поле поиска
                                                                   ~ - обесценить значимость слова, влияет на порядок следования результатов
                                                                   * - любое кол-во любых символов
                                                                   "" - в двойных кавычках точное соответсвие строке поиска
